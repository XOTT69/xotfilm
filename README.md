# mono — Тренажер оператора

Симулятор інтерфейсу оператора служби підтримки Monobank для навчання співробітників.

## Як запустити

Відкрийте файл `index.html` у браузері (подвійний клік або перетягніть у вікно браузера).

Або через термінал:
```bash
open index.html
# або
python3 -m http.server 8080
# потім відкрийте http://localhost:8080
```

## Функціонал

- **Пошук клієнтів** — за номером телефону або ім'ям
- **Інформація про клієнта** — баланс, статус, тип акаунту
- **Транзакції** — історія операцій клієнта
- **Картки** — перегляд карток та балансів
- **Чат** — симуляція листування з клієнтом (з автовідповіддю)

## Дані

Усі дані фейкові, створені для тренування. 5 тестових клієнтів з різними сценаріями.

## Дизайн

Стилізація в дусі Monobank: чорний фон, лаймовий акцент (#B8FF3C), мінімалістичний інтерфейс.
